CONCEPT

かわいい家計簿ではなく、やさしい作戦ボード。

ハネミーは「何にいくら使ったか」を細かく追うよりも、「今月をどう過ごすか」を先に決めるためのアプリです。生活費を守りながら、楽しみも諦めない。そのための小さな作戦ボードを目指しています。

今月の残りを確認

開いた瞬間に、今月あとどれくらい安心して使えるかを確認できます。

ざっくり入力

細かい店名やメモではなく、使った分だけ作戦ボードに反映します。

カテゴリ別に見る

生活費・楽しみ用・やりたいことを一覧で確認し、使用率も見られます。

今月の準備

収入、固定費、生活費、楽しみ用を決めて、今月の方針を作ります。

画像で保存

家族に見せたり、自分用に今月の作戦を画像として残せます。

公開βとして改善中

現在は公開βです。軽く使える形を保ちながら、少しずつ磨いています。

HOW TO USE

使い方は、4ステップ。

1今月の準備

収入と固定費を入れます。

2作戦を決める

生活費と楽しみ用を分けます。

3ざっくり入力

使った分だけ軽く記録します。

4残りを見る

今月の現在地を確認します。

INSTALL

スマホのホーム画面に追加できます。

毎日開くなら、ブラウザのブックマークよりホーム画面追加が便利です。アプリのようにすぐ開けます。

iPhone Safariでアプリを開く → 共有ボタン → ホーム画面に追加
Android Chromeでアプリを開く → メニュー → ホーム画面に追加 / アプリをインストール
